Prerequisites for build

  • Install exe4j from sbin/exe4j_windows_4_0_3.exe.
  • Install Advanced Installer from sbin/advinst.msi.
  • Get a runtime license for JNIWrapper and copy it to etc/keys/jniwrap.lic.
  • Get a license for yDoc and copy it to etc/keys/ydoc.license.txt.
  • Get a license for exe4j. Start the program and click the Change License button to apply it.
  • Get a license for Advanced Installer. Click on Help/Register to apply it.
  • See etc/api/overview.html for more information.

How To Build The Project

  • Open a command prompt window.
  • Change to the project's base directory.
  • Type 'build clean' and hit Enter.
  • Type 'build all' and hit Enter.

Notes

  • Always use build.cmd to perform builds.
  • Use the "build -?" command to get a list of available Ant targets.
  • API docs contain a listing of all third-party libraries and components used to build and run the application, including licensing information.
  • Please update etc/api/overview.html when libraries or tools are added or updated.
  • All dependencies are free and open-source except for JNIWrapper, yDoc, exe4j, Advanced Installer, which are commercial.
